## 核心要点（30 秒）
- 目标：车机生态 App 跨品牌发版回归自动化（一次写、多品牌跑、自动出报告）。
- 栈：FastAPI+SQLite（后端）/ 免构建分层静态前端 / pytest+uiautomator2（框架）/ Allure 真报告。
- 数据主线：功能用例(分母·官方Excel导入) → 脚本(分子) → 任务 → 执行(真机⇄模拟) → 报告(Allure) → 覆盖率看板。
- **平台兼容多套框架**：我们的 framework / Media_automation(ADB-XML) / Zcode(u2)，上传 zip 自动识别、各写一个适配器接入。
- 覆盖率：业务用例覆盖率（全量 + P1/P2 双口径）+ 预估节省人力；自动化状态由框架映射自动判定。
- ⚠️ 本平台端口 **8770**；8765 是并行的 Codex 平台，不动。
